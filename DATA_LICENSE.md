# Data And Artifact License Notes

This repository contains code, documentation, generated figures, and derived result tables.

## Code

Source code is released under the MIT License. See `LICENSE`.

## Derived Artifacts

The derived CSV tables, markdown reports, and generated figures in `results/`, `docs/`, and `figures/` are provided for scholarly inspection and reuse with attribution to the repository and associated manuscript.

## Raw Data

Raw station archives are not redistributed in this repository. They should be obtained from the original public providers, including GHCN-D, ECA&D, DWD, MeteoSwiss, AEMET, ECCC, INMET, and the Hong Kong Observatory, under each provider's own access terms.

The station identifiers, regional assignments, descriptors, degradation summaries, and decision-layer outputs included here are derived artifacts intended to support reproducibility of the reported analyses.
